# TASK
Написать многопоточную программу на C, использующую библиотеку pthreads,
для параллельной сортировки целочисленного массива любым методом.
* Параметры командной строки: количество потоков (одно натуральное число).
* Входные данные: не сортированный массив из целых чисел, разделенных
пробелом, в стандартный поток ввода.
* Выходные данные: отсортированный массив из целых чисел, разделенных
пробелом, в стандартный поток вывода.
#### Пример запуска:
input
```
echo 3 2 1 | task1 12
```
output
```
1 2 3
```
